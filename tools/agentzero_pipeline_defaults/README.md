# AgentZero Phone Shorts Pipeline Defaults

These files are copied into `/app/work_dir/assets/agentzero_pipeline/` on the server.

- `pipeline_config.example.json` is the editable SOP/config template.
- `audio/brand/` is where users place their own licensed SFX and music.
- `logo/` is where users place their own `logo.png`.

No media assets are included in this public package. Bring your own licensed sounds, logo, and music bed.
